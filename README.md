# KOReader User Patches

Патчи для KOReader на PocketBook 632. Все файлы кладутся в папку:
```
PB632/applications/koreader/patches/
```

## Содержимое

| Файл | Описание |
|---|---|
| `21-fallback-font_RC6.lua` | 5 дополнительных резервных шрифтов через меню шрифтов |
| `22-fb2zip-smart-current_RC5.lua` | Открывает `.fb2.zip` через автоматическую распаковку во `/tmp`, сохраняет историю и прогресс |
| `23-epub-fix-toc_RC3.lua` | Исправляет несоответствие spine/toc.ncx в epub **без лишней распаковки** — проверка через `unzip -p`, полная распаковка только при необходимости |
| `8-clear-crashlog-on-exit.lua` | Очищает `crash.log` при чистом выходе из KOReader |

## Правила нумерации патчей

Согласно [официальной документации KOReader](https://github.com/koreader/koreader/wiki/User-patches):

- `2-*.lua` — выполняются после инициализации UI (при старте)
- `8-*.lua` — выполняются перед выходом из программы и перед сохранением настроек

## Установка

1. Скопируйте нужные `.lua` файлы в `PB632/applications/koreader/patches/`
2. Перезапустите KOReader
3. Проверьте `crash.log` на наличие ошибок при загрузке патчей
